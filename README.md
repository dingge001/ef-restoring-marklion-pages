# ef-restoring-marklion-pages

根据标记狮（MarkLion）导出文件进行页面 1:1 还原、像素级收口、字体映射、图层对齐与交互重建的 OpenClaw 技能。

## 功能特性

- **项目技术栈自动检测** - 分析项目使用的框架和依赖
- **标记狮快照对比** - 对比设计稿与已实现页面的差异
- **数据校验** - 验证标记狮导出数据的完整性
- **视觉回归报告** - 生成像素级对比报告
- **色彩还原** - 高精度色彩保真处理
- **共享布局提取** - 识别和复用公共布局模块
